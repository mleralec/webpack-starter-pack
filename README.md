# Webpack starter pack

> Quickly initiate a JS project

### Installation

```bash
git clone https://github.com/mleralec/webpack-starter-pack.git
cd webpack-starter-pack
yarn install
```

### Available scripts

```bash
yarn start      # Run dev server on port 9000
yarn dev        # Build without optimizations in /dist folder
yarn prod       # Build with optimizations in /dist folder
yarn clean      # Clean /dist folder
yarn serve:dev  # Run yarn dev and server /dist folder
yarn serve:prod # Run yarn prod and server /dist folder
```
